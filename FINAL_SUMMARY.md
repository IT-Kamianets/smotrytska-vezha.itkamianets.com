| **Світлий текст** | `#ecf0f1` | Footer, контрастний текст |
| **Фон** | `#f8f9fa` | Основний фон сайту |

### Адаптивність (Mobile-first)
```css
/* Desktop (> 1024px) */
- Full layout
- 3-колонна сітка
- Всі елементи видимі

/* Tablet (768px - 1024px) */
- 2-колонна сітка
- Зменшені елементи
- Мобільне меню

/* Mobile (< 768px) */
- 1-колонна сітка
- Burger меню
- Оптимізовані шрифти
```

- ✨ `slideUp` - вступ заголовків
- 🔍 `zoomIn` - увеличение фону
- 🎯 `bounce` - стрілка скролу
- 📍 `scale` - картки при наведенні
- 🎨 `translateY` - ефект тиску кнопок
- 🌈 `transition` - гладкі переходи (0.3s)

## ✨ КЛЮЧОВІ ОСОБЛИВОСТІ

✅ **Семантична HTML5**
- `<header>`, `<section>`, `<footer>`
- `role`, `aria-label` для доступності
- Правильна структура заголовків

✅ **Mobile-first дизайн**
- Адаптивний під 320px - 2560px
- Touch-friendly кнопки (45px мін.)
- Оптимізовані зображення

✅ **Без сторонніх UI бібліотек**
- Тільки Bulma CSS + кастомний CSS
- Немає Bootstrap, Material, etc.
- ~200KB CSS (скопреса)

✅ **Плавна навігація**
- Скрол до секцій без перезавантаження
- `scroll-behavior: smooth`
- Активна навігація при скролі

✅ **Реактивні компоненти**
- Angular Signals: `signal()`, `update()`
- Динамічні дані з mock-масивів
- Два способи передачі даних

✅ **Оптимізація продакшену**
- Tree-shaking для видалення мертвого коду
- Lazy loading для зображень
- Async компоненти
