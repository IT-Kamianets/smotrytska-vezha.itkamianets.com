## 🚀 ШВИДКИЙ СТАРТ

### 1. Встановлення (першого разу)
```bash
cd hotel-landing
npm install
```

### 2. Запуск розробницького сервера
```bash
ng serve
```

### 3. Відкрити браузер
Перейти на **http://localhost:4200**

### 4. Редагувати файли
Сайт автоматично оновлюється при змінах!

### 5. Збудувати для продакшену
```bash
ng build --configuration production
```

Файли буде у папці **dist/hotel-landing/**
