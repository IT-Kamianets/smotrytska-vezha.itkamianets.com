# smotrytska-vezha.itkamianets.com

Репозиторій містить сайт готелю **Смотрицька Вежа** у Кам'янці-Подільському.

## Структура

- `hotel-landing/` — основний Angular-проєкт, який будує і запускає сайт.

## Запуск локально

```bash
cd hotel-landing
npm install
npm start
```

Після запуску сайт буде доступний за адресою:

```text
http://localhost:4200/
```

## Основні технології

- Angular 21
- Bulma CSS
- Express (SSR-підтримка)
